# Security Policy

## Reporting a Vulnerability

If you believe you have discovered a security issue within this documentation, please report it responsibly.

Please do not disclose potential issues publicly before they have been reviewed.

---

## Supported Versions

This guide focuses primarily on modern MikroTik RouterOS releases intended for production environments.

Older versions may no longer reflect current security recommendations.

---

## Scope

This repository contains documentation only.

It does not include production-ready RouterOS configurations or software.

---

## Contact

Security-related questions or documentation improvements are welcome through GitHub Issues or Pull Requests.
