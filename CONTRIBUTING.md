# Contributing

Thank you for your interest in improving this project.

This repository is intended to provide a high-quality reference guide for securing MikroTik RouterOS in business environments.

---

## Before Contributing

Please consider the following principles:

- Prioritize accuracy over complexity.
- Follow enterprise security best practices.
- Keep recommendations vendor-neutral whenever possible.
- Explain *why* a recommendation exists, not only *how* it can be implemented.
- Avoid customer-specific configurations.

---

## Documentation Style

When contributing documentation:

- Use clear technical English.
- Keep paragraphs concise.
- Prefer explanations over configuration examples.
- Follow existing chapter structure.
- Use Markdown formatting consistently.

---

## Reporting Issues

If you discover:

- inaccurate information;
- outdated recommendations;
- broken links;
- security concerns;
- documentation errors;

please open an Issue describing the problem.

---

## Pull Requests

Pull Requests are welcome.

Please ensure that:

- information is technically correct;
- formatting follows the existing style;
- documentation remains consistent;
- recommendations are supported by current best practices.

---

Thank you for helping improve this guide.
